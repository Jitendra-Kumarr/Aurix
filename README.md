## 📖 Overview

**Aurix** is a smart alarm application designed to eliminate the habit of snoozing.

Unlike traditional alarm apps that allow users to stop or snooze alarms with a single tap, **Aurix challenges users to complete a real-world task before the alarm can be dismissed.**

The application uses **AI-powered object recognition** to verify that the required task has actually been completed.

---

## 💡 The Problem

Millions of people struggle with:

- 😴 Oversleeping
- ⏰ Hitting the snooze button repeatedly
- 📅 Poor morning routines
- 📉 Reduced productivity

Traditional alarms are easy to ignore.

Aurix changes that.

---

# 🚀 How It Works

1. User sets an alarm.
2. Alarm rings.
3. Aurix generates an object-based challenge.
4. User captures an image using the camera.
5. AI verifies the object.
6. Alarm stops only after successful verification.

---

## 🎯 Example Challenges

Aurix may ask users to capture images such as:

- 🚰 Running Tap Water
- 🪥 Toothbrush
- ☕ Coffee Mug
- 🌳 Tree
- 📖 Book
- 🚪 Door
- 👟 Shoes
- 💧 Water Bottle
- 🌞 Window with Sunlight

Every challenge requires the user to physically get out of bed.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/USERNAME/Aurix.git
```

Move into the project

```bash
cd Aurix
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npx expo start
```
